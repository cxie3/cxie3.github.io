Web Evaluations

Listed below are two websites that are critiqued by CRAP. CRAP stands for:

  • Contrast
  • Repetition
  • Alignment
  • Proximity

Two websites broken down by the CRAP principle.


Good Website: Spotify Design

Spotify Design is a website that shows a visual of the the design community for spotify.

  • Contrast: There is a variety of colors that make the website very vibrant, but the text is is always contrasting the background color and it allows the user to see the text clearly.

  • Repetition: The website has clear sections divided through the website as the user scrolls down. The division can be seen with a background color change.

  • Alignment: There are very clear margins from top, down, left and right that makes the page look well rounded.

  • Proximity: The website has a lot of attention to proximity allowing enough space for all the content to make sure the user feels comfortable with the space.

Bad Website: Craigslist

Craigslist is a website that allows users to view and post advertisements for almost anything.

  • Contrast: The website has the default colors for links and backgrounds which make the overall first impression very plain.

  • Repetition: Every is very plain looking so the colors, alignment, column placements are repeating.

  • Alignment: The links are aligned to the left of the columns. There is a margin from the left and right which push the columns to the center a bit.

  • Proximity: All the content in the website have a very close proximity making the website feel cramped.
